HTML Primer

In order for you to write notes on this website, you're probably going to need to learn the basics of HTML. Try opening a .html file in gedit. Notice how everything is made up of words surrounded by angle brackets, for example <p>? These are called tags and usually they come in pairs: <b>tags</b>. Here, the first tag says 'start making things bold', whereas the second tag - with the / says 'stop making things bold'. Similarly, <p>Here is a paragraph</p> has a tag that says 'start the paragraph' and 'end the paragraph':

Here is a paragraph

The following is a quick reference to help you make some damn fine-looking notes!

Tags Function Example

<i>Words</i> Makes things italicised Words
