asdfSometimes when you're working on a project you'll lose your thread or train of though. "Where am I?" you wonder, trying to remember what branch you're on, what you just did or what you're working on at the moment.

When this happens you'll get an irresistable urge to type 'ls' over and over again - but wait! You can do more!

There are a lot of commands that show you the state of your git project to help you get a hold of where you're up to and are very handy. Here's a short list of commands that should be able to show you (almost) all the information you need. Feel free to try them out whenever and wherever and as often as you like throughout this workshop to see how your git repo changes #informationispower

  • git status
  • git branch
  • git log
  • git show-ref
  • git remote -v
  • git config -l
  • git rev-list HEAD
  • git help
  • git help <command> (e.g. git help commit)
